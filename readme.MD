Simple app to manage UTA transactions.  
Here is th ways to use it:
1. Uta API:
   (endpoint "/all")  
To use API you need access to official UTA API.  
To do this you need clientSyncId, userName and password.  
This credentials are only available when you are UTA bussines client and ask for them.  

2. CSV: (endpoint "/csv" and "/fuel-usage)  
To use this endpoint you need to have csv files converted by utaWinImport app from UTA files.  
Also need to have access to FTP to your account in UTA client portal.
